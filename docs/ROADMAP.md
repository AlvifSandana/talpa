# Roadmap

## v0.x (MVP)
- Safety core: path validation, blocked paths, symlink guard, operation log
- `clean`, `analyze`, `purge`, `status`
- `completion`, `update`, `remove`
- JSON output stability (`--json`)

## v1.0
- `uninstall` with XDG leftover cleanup
- `installer` cleanup
- `optimize` safe actions with preflight checks
- Full package manager integrations

## v1.x+
- Ruleset plugins (YAML/JSON)
- Profiles (Dev/Desktop/Minimal)
- Optional launcher integrations
